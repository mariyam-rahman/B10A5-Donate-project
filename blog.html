Important FAQ

What is the Document Object Model (DOM)?

It's a tree of a webpage elements that can be changed with JavaScript.

How do you select an element from the DOM?

I will use document.getElementById() for an ID, document.getElementsByClassName() for a class, document.getElementsByTagName() for a tag, document.querySelector() for the first match, or document.querySelectorAll() for all matches.

What is event delegation in the context of the DOM, and why is it useful?

It lets a parent handle events for its children, so i dont need to add event listeners to each child.

How do you manipulate an element's attributes and styles using the DOM?

I can use element.setAttribute() for attributes and element.style for styles.
